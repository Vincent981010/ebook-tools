/** * 這是整合了您原本繁簡轉換邏輯與「自訂詞彙替換」的完整核心 * 使用方式：將此程式碼放入您原本的 script 區塊中，或替換對應的函式 */ // 1. 在既有的變數宣告區加入替換規則容器 (建議放在 script 最上方) const rules = [ { find: "欲替換的詞A", replace: "替換後的詞A" }, { find: "欲替換的詞B", replace: "替換後的詞B" } // 您可以依此格式增加更多規則 ]; // 2. 修訂後的核心 EPUB 轉換邏輯 (完整保留您原版的編碼與連結處理) async function processSingleEpub(file, converter) { const zip = await new JSZip().loadAsync(file); const newZip = new JSZip(); if (zip.files['mimetype']) { const mimeContent = await zip.file('mimetype').async('string'); newZip.file('mimetype', mimeContent); } for (const [path, entry] of Object.entries(zip.files)) { if (entry.dir || path === 'mimetype') continue; const isTextFile = path.match(/\.(html|xhtml|htm|ncx|opf|xml|txt|css)$/i); const newPath = converter(path); if (isTextFile) { let content = await entry.async("string"); // --- [新增功能] 字詞抽換邏輯 --- rules.forEach(rule => { content = content.split(rule.find).join(rule.replace); }); // ------------------------------ // 處理網址編碼 content = content.replace(urlPattern, (match, attr, quote, urlVal) => { try { return `${attr}=${quote}${decodeURIComponent(urlVal)}${quote}`; } catch (err) { return match; } }); // 執行 OpenCC 繁簡轉換 content = converter(content); // 修正編碼宣告 content = content.replace(xmlEncPattern, "$1utf-8$2"); content = content.replace(metaEncPattern, "$1utf-8$2"); newZip.file(newPath, content); } else { // 非文字檔直接複製，確保圖片結構不損壞 newZip.file(newPath, await entry.async("blob")); } } return await newZip.generateAsync({ type: "blob", mimeType: "application/epub+zip" }); }
